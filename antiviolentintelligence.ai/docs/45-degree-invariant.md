# 45° Invariant

The 45° relationship represents a balanced diagonal between two orthogonal components.

Instead of collapse into a single axis, both components contribute equally.

## Geometric definition

Consider a right triangle with equal legs:

\[
a = 1,\quad b = 1
\]

The diagonal becomes

\[
c = \sqrt{(1^2 + 1^2)}
\]

This produces a 45° angle relative to each axis.

## Cosine relationship

Cosine similarity expresses the same geometry.

\[
\cos(\theta)=\frac{\langle u,v\rangle}{\|u\|\|v\|}
\]

For a balanced diagonal:

\[
\theta = 45^\circ
\]

\[
\cos(45^\circ)=\frac{1}{\sqrt{2}}
\]

This means each component contributes equally to the resulting direction.

## Why this matters

Many stable systems operate through **balanced contributions**, not dominance of one axis.

Examples include:

- vector projections
- machine learning embeddings
- signal processing
- physical force decomposition
- geometric symmetry

A 45° relationship keeps both components active.

## Vector example

Let

\[
u = (1,0)
\]

\[
v = (1,1)
\]

Then

\[
\cos(\theta)=\frac{1}{\sqrt{2}}
\]

which corresponds to a 45° relationship.

## Interpretation

The 45° diagonal can be understood as:

- balanced projection
- symmetric contribution
- stable relation between components

It is neither collapse into one direction nor cancellation between opposing directions.
