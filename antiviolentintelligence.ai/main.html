We analyze density over the progression \(n\equiv 5\pmod{6}\) under two filtering regimes: primorial (multiplicative) sieves and finite modular constraints. We prove and verify that primorial filtering yields densities \(D_{k}=\prod_{p\geq 5}(1-1/p)\) that decay to zero, whereas finite modular exclusions yield densities \(D=\prod_{i}(1-1/m_{i})\) that remain bounded, including the exact case \(24/25\). 1. Domain Definition

Let \[ S_{L} = \{ n \leq L : n \equiv 5 \pmod{6} \}. \] Normalized density: \[ D(L) = \frac{|S_{L}|}{L/6}. \]

2. Primorial Sieve (unbounded multiplicative filtering)

Let \(P_{k} = \prod_{i=1}^{k} p_{i}\) be the primorial of the first \(k\) primes. Define

S_{k,L} = \{ n \leq L : n\equiv 5\pmod{6},\ \gcd(n,P_{k})=1 \}.

Only primes \(p \ge 5\) contribute to the constraint, giving density factor

D_{k} = \prod_{\substack{p \ge 5 \\ p \mid P_{k}}} \left(1 - \frac{1}{p}\right).

As \(k \to \infty\), the infinite product over primes \(p \ge 5\) satisfies

\prod_{p \ge 5} \left(1 - \frac{1}{p}\right) = 0.

Result: Primorial filtering forces density to collapse to zero — unbounded constraint accumulation eliminates admissible states asymptotically.

3. Structured Modular Constraint (finite, bounded)

Let \(m\) be such that \(\gcd(m,6)=1\) (so the modulus respects the residue class modulo 6). Define the excluded set:

S_{m,L} = \{ n \leq L : n\equiv 5\pmod{6},\ n \not\equiv 0 \pmod{m} \}.

The asymptotic density becomes

D = 1 - \frac{1}{m}.
✔ Exact constant: \(m = 25\)   →   \( D = 1 - \frac{1}{25} = \frac{24}{25} = 0.96.\)
24/25 = admissible fraction under +5 constraint layer (p=5 isolation).

4. Multi‑Constraint System (finite, coprime moduli)

If \(m_1,\dots,m_r\) are pairwise coprime and \(\gcd(m_i,6)=1\) for each \(i\), the combined exclusion yields

D = \prod_{i=1}^{r} \left(1 - \frac{1}{m_i}\right).

Example: \( D = \frac{24}{25} \cdot \frac{48}{49} \), which remains strictly positive and bounded away from zero.
