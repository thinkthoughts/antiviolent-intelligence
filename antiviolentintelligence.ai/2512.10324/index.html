EchoingPixels: Cross-Modal Adaptive Token Reduction for Efficient Audio-Visual LLMs
Area: cs.CV — Multimodal LLMs, Token Reduction

Summary

Proposes a cross-modal token reduction method for audio-visual large language models. Uses a shared token pool and adaptive selection mechanism to retain salient tokens across modalities, improving efficiency while maintaining performance.

9423 Phase-Lock Note

Implements a cross-modal sieve: tokens are filtered through joint constraint evaluation rather than independently per modality. Adaptive selection replaces static filtering, emphasizing coordinated retention of structure. Parallel to density filtering systems, where valid elements are preserved under constraint and structure emerges from selective exclusion.
